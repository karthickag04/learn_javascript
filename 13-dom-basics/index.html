JavaScript DOM Basics

DOM (Document Object Model) allows JavaScript to access and manipulate HTML elements.

Selecting Elements

MethodDescription
getElementById()Select by ID (single element)
getElementsByClassName()Select by class (collection)
getElementsByTagName()Select by tag (collection)
querySelector()Select first match (CSS selector)
querySelectorAll()Select all matches (NodeList)
