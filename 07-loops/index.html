JavaScript Loops

Loops are used to repeat a block of code multiple times.

Types of Loops:

Loop Description Use Case
for Repeats code a specific number of times When you know how many iterations
while Repeats while condition is true When iterations depend on a condition
do...while Executes at least once, then repeats while true When you need at least one execution
for...of Iterates over values of an iterable Arrays, strings
for...in Iterates over properties of an object Objects

Open the browser console (F12) to see the JavaScript output.
