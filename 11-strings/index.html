JavaScript Strings

Strings are sequences of characters used to represent text.

Common String Methods:

MethodDescription
lengthGet string length
toUpperCase()Convert to uppercase
toLowerCase()Convert to lowercase
indexOf()Find position of substring
slice()Extract portion of string
substring()Extract between indices
replace()Replace text
split()Split into array
trim()Remove whitespace
includes()Check if contains text
startsWith()Check if starts with
endsWith()Check if ends with

Open the browser console (F12) to see the JavaScript output.
