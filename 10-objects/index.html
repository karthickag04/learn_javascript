JavaScript Objects

Objects are collections of key-value pairs. They are the foundation of JavaScript.

Key Concepts:

  • Objects store data as properties (key-value pairs)
  • Objects can contain methods (functions)
  • Properties are accessed using dot notation or bracket notation
  • Objects are mutable (can be changed)
  • Objects are passed by reference

Object Syntax:

let person = {
    name: "John",          // Property
    age: 30,               // Property
    greet: function() {    // Method
        return "Hello!";
    }
};

Open the browser console (F12) to see the JavaScript output.
