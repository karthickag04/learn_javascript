Console and Output Methods

How to view output?

Press F12 to open Developer Tools, then click on the Console tab.

Topics Covered:

  • console.log() - Print messages
  • console.table() - Display data in table format
  • console.clear() - Clear console
  • console.warn() - Warning messages
  • console.error() - Error messages
  • alert() - Popup message
  • prompt() - User input popup
  • document.getElementById().innerHTML - Write to HTML
