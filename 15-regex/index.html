JavaScript Regular Expressions

Regular Expressions (Regex) are patterns used to match character combinations in strings. They are powerful tools for searching, validating, and manipulating text.

Regex Syntax Quick Reference

PatternDescriptionExample
.Any character except newlinea.c matches "abc", "a1c"
\dDigit (0-9)\d\d matches "12", "99"
\wWord character (a-z, A-Z, 0-9, _)\w+ matches "hello"
\sWhitespacea\sb matches "a b"
^Start of string^Hello starts with "Hello"
$End of stringworld$ ends with "world"
*0 or moreab* matches "a", "ab", "abb"
+1 or moreab+ matches "ab", "abb"
?0 or 1colou?r matches "color", "colour"
{n}Exactly n times\d{4} matches "2023"
[abc]Character class[aeiou] matches vowels
(abc)Capturing group(\d+)-(\d+) captures numbers

Flags

FlagDescription
gGlobal - match all occurrences
iCase-insensitive
mMulti-line
sDotall - dot matches newlines
