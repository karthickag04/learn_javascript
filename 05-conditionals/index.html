Conditional Statements

Conditional statements are used to make decisions in code based on conditions.

Types of Conditionals:

  • if - Execute code if condition is true
  • if...else - Execute one block if true, another if false
  • if...else if...else - Multiple conditions
  • switch - Match value against multiple cases
  • ? : - Ternary operator (shorthand)

Basic Syntax:

if (condition) {
    // code to execute if condition is true
} else if (anotherCondition) {
    // code if first condition false, this true
} else {
    // code if all conditions are false
}
