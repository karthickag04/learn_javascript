Array Iteration Methods

Array methods provide powerful ways to iterate, transform, and manipulate arrays.

Key Iteration Methods:

Method Purpose Returns
forEach() Execute function for each element undefined
map() Transform each element New array
filter() Keep elements that pass a test New array
reduce() Reduce array to single value Single value
find() Find first matching element Element or undefined
some() Check if any element passes test Boolean
every() Check if all elements pass test Boolean

Open the browser console (F12) to see the JavaScript output.
